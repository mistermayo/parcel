Shipping Calculator
===================

Product Description
-------------------

Simple web app for calculating the cost of shipping a package based on the dimensions and weight.

Setup Instructions
------------------

* Download project folder
* Go into terminal and type "gem install bundler"
* Type "bundle"
* Type "ruby app.rb"
* Go to localhost:4567/
